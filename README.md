# CSS Checkbox & Radio Collection

A single-page project that showcases **30 unique styles for checkboxes and radio buttons** built using **pure HTML and CSS**.

Each style includes a **checkbox and a matching radio button**, demonstrating different UI techniques such as gradients, glow effects, glass styles, and animated interactions.

The project is designed as a **learning resource and UI inspiration** for styling form controls with modern CSS.

---

## ✨ Features

* 🎨 30 unique checkbox styles
* 🔘 30 matching radio button styles
* ⚡ Pure CSS (no JavaScript)
* 📱 Responsive grid layout
* ♿ Accessible inputs with proper labels
* 🌙 Dark mode support
* 🧩 Easy to copy and reuse in forms

## 🧠 How It Works

* Real `<input type="checkbox">` and `<input type="radio">` elements are used for proper **accessibility and semantics**.
* The default browser appearance is removed using:

```css id="7x9sda"
appearance: none;
```

* Each style pair uses:

* `.cb-1` → `.cb-30` for checkboxes

* `.rd-1` → `.rd-30` for radio buttons

Each variant applies different visual effects using modern CSS.

## 🎨 CSS Techniques Used

Some styles demonstrate:

* Gradient borders
* Glassmorphism
* Neon glow effects
* Embossed / inset styles
* Conic gradient stripes
* Notched shapes
* SVG icons via data-URI
* Clip-path shapes
* Rotation and scale animations
* Custom focus rings

## ♿ Accessibility

Accessibility was considered in the design:

* Inputs remain **focusable**
* Labels provide **large click targets**
* `:focus-visible` used for keyboard focus
* Good **color contrast**
* Works with screen readers
